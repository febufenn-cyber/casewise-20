# Phase 3 — Allegation, Response, Evidence, and Contradiction Mapping

Phase 3 converts the reviewed Phase 2 matter graph into a source-linked allegation-response-evidence workspace.

The phase preserves these distinctions:

- an allegation is an attributed proposition, not an established fact;
- a response must be separately sourced and may admit, deny, partially admit, fail to answer, remain ambiguous, or contradict another position;
- evidence may support, contradict, contextualize, or merely be cited for a proposition;
- a contradiction is a review candidate until a lawyer confirms or explains it;
- missing means not located within processed scope, not nonexistent;
- generated prose and filing-ready drafting remain outside Phase 3.

Implementation increments:

1. allegation register;
2. response mapping;
3. evidence and claim-support relationships;
4. contradiction and missing-information registers;
5. reviewed matrix snapshots and exit validation.
