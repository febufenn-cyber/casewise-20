# Response mapping contract

A response is a separately sourced statement linked to one or more allegations. Its classification does not alter the allegation's source text.

Supported classes are admitted, denied, partially admitted, not specifically answered, ambiguous, and contradicted elsewhere. A response may address the allegation fully, partially, or unclearly.

“Response not located” is not a response class and never means “not denied.” It is a reviewed search-coverage result that records which logical documents were searched and whether that search was complete.

Quoted, reported, and inferred responses remain visibly distinguished from direct statements. Corrections preserve review lineage and stale dependent objects.
