# Phase 1 incident-response baseline

A named production owner is required before confidential pilots.

Immediate actions: isolate affected queues and processors; revoke Worker, Supabase, R2, and processor secrets; disable affected matters without destroying investigative evidence; identify scope from metadata-only audit events; preserve relevant system evidence without copying matter content into ordinary tickets; escalate to providers; and decide customer or regulatory notification with qualified counsel.

Every incident must create a regression test or evaluation fixture before processing is restored.
