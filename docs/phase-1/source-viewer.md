# Source viewer contract

Canonical page identity is `uploaded_file_id + pdf_page_index`; printed, logical-document, exhibit, and paragraph labels are display metadata. Highlights use normalized 0–1 coordinates from the top-left after canonical rotation and remain tied to one render version. The viewer discloses file hash, page systems, extraction method/version, warnings, stale state, quoted text, and review status. Page images use short-lived private capabilities.
