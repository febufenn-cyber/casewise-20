# Casewise Phase 1 implementation

Phase 1 adds the secure evidence-custody substrate beneath the Phase 0 product constitution.

It includes a Hono Worker API, Supabase RLS schema, R2-backed immutable evidence storage, durable queued processing, an isolated native PDF/OCR processor, a source viewer, audit events, coverage accounting, and deletion propagation.

Start with [`docs/phase-1/README.md`](docs/phase-1/README.md).

Phase 1 intentionally contains no legal summary, issue extraction, authority research, or response drafting.
